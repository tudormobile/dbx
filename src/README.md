# Source Projects

This folder contains the .NET source code, unit tests, and sample host for dbx.

## Current Layout

- `dbx/`: core library project.
- `dbx.tests/`: unit tests for the core library.
- `host/`: sample host for testing.
- `dbx.slnx`: solution file that currently hosts the library, host app, and tests.

Documentation may be found in the repository root readme file.
